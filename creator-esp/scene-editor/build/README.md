# Construyendo Tu Escena

Usa las herramientas de construcción del Scene Editor para colocar objetos, configurar componentes y crear los elementos visuales de tu escena. Esta sección cubre todas las características esenciales de construcción.

## Lo que Aprenderás

* Agregar y configurar componentes
* Importar ítems y assets personalizados
* Mostrar NFTs en tu escena

El Scene Editor proporciona un conjunto poderoso de herramientas para construir visualmente tus experiencias de Decentraland.
